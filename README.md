# Running the app
    1) Start database
        $ cd ~/mongo/bin
        $ ./mongod --dbpath ~/mongo-data
    2) npm install 

    3) Start server 
        $  npm start OR npm run webpack
        
    4) Build bundle 
        $ npm run build
        
# Test
    - Testing server/API
        $ npm run testServer
    - Testing client/React
        $ npm run testClient
